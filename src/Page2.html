UI Designers

UI designers work on the interactive parts of apps, websites, and devices. They need to be able to collaborate with others, have color theory, typography, empathy, and design pattern skills. These attributes of a UI designer will make them much more successful. Having these skills will help them in their career. Some essential terms used in UI design are below.

  • Typography
  • Color Theory
  • Prototype
  • Wireframe
  • Breadcrumb
  • Accessibility
  • Affordance
